❤️ Love Score Calculator (PyScript)

The Python script calculates the love score for "Angela Yu" and "Jack Bauer" and displays the result below.

# Your original Python code goes here def calculate_love_score(name1, name2): name3 = name1 + name2 # Count occurrences of letters in 'true' count1 = 0 for i in name3: if i.lower() in "true": count1 += 1 # Count occurrences of letters in 'love' count2 = 0 for i in name3: if i.lower() in "love": count2 += 1 # Concatenate counts and convert to integer score concatinate = int(str(count1) + str(count2)) # The print() function will output to the element with id="output-div" print(f"The Love Score is: {concatinate}") calculate_love_score("Angela Yu", "Jack Bauer")
